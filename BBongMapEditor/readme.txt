맵에디터

	스테이지별로 나올 적이나 사물등을 바꿔 줄수 있는 에디터이다.

	자신이 나올 적, 사물을 바꿀수 있기 때문에 적당한 난이도로 조정도 할수 있도록 해준다.


사용법 : 

	먼저 각 스테이지 bmp 파일을 로드한다.

	그후 적이나 사물이 등장하길 원하는 곳에 클릭을 하면 다이알로그 박스가 뜬다.

	그중 좌표는 적이나 사물이 나올 좌표이고 에너지는 적일 경우에는 에너지가 되고 사물일 경우에는 

	0일 경우에는 터지는 사물 0이 아닌 값일때는 터지지 않는 사물이 등장하게 된다.

	그리고 정지화면 등장시기는 0 일때는 화면이 그곳까지 움직였을 경우 등장을하게 되고 

	0 아닌 경우에는 일단 화면 스크롤이 정지가 되고 등장시기가 같은 모든 적 사물이 등장하게 된다.

	적타입에는 등장시키고 싶은 적이나 사물을 선택하고 아이템에는 사물이 등장할 경우 

	그 사물이 터졌을 때 나올 아이템을 선택하면 된다.

	그렇게 해서 나올 적 사물을 다지정하고 난후 저장을 하는데

	스테이지1 은 stage1_a.dat 파일로
	스테이지3 은 stage3_a.dat 파일로
	스테이지4 은 stage4_a.dat 파일로
	스테이지5 은 stage5_a.dat 파일로 저장하면 된다.

	이렇게 저장된 dat 파일들을 뽕2002 게임 폴더로 복사 시켜 넣으면 자신이 등장시킨 적 사물 들을

	가지고 게임을 즐길수 있다.



※ 주의 
	스테이지별로 마지막에 등장하는 적은 꼭 등장시기 가 0 이 아니어야 한다.

	만약 0일 경우 스크롤이 계속 되어 화면이 깨진다.
